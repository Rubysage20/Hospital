# 🏥 Hospital Management System – Java OOP Project

Welcome to the Hospital Management System project! This Java-based application is designed to manage hospital operations efficiently, utilizing object-oriented programming principles. It allows for the management of patients, doctors, employees, and clinics through a modular and extensible codebase.

---

## 📂 Project Structure

- **Clinic.java** – Manages clinic-related information and operations.
- **Doctor.java** – Handles doctor profiles and their associated data.
- **Employee.java** – Manages general hospital staff information.
- **Patient.java** – Handles patient records and related functionalities.
- **Person.java** – Serves as a superclass for shared attributes among individuals.

---

## 🛠️ Technologies Used

- **Programming Language:** Java
- **Concepts:** Object-Oriented Programming (Inheritance, Encapsulation, Polymorphism)
- **Development Environment:** Eclipse
